📚 API 使用说明

POST /api/generate-suggestions
生成3-5个行动建议，每个建议包含表情符号、标题和内容
📤 请求示例
{ "input": "我想要提高工作效率" }
📥 返回示例
{ "success": true, "message": "建议生成完成", "suggestions": [ { "emoji": "⏰", "title": "时间管理优化", "content": "使用番茄工作法，25分钟专注工作..." } ] }
🧪 在线测试
POST /api/get-link-content
获取链接的网页内容，通过飞书表格自动解析和总结网页信息
📤 请求示例
{ "url": "https://www.example.com" }
📥 返回示例
{ "success": true, "message": "内容获取成功", "content": { "标题": "示例网站", "内容": "网页的主要内容..." } }
🧪 在线测试
POST /api/generate-feedback
基于执行总结数据生成进展反馈，包含执行状态反馈和改进建议两部分
📤 请求示例
{ "execution_summary": { "weekly_executions": 5, "average_rating": 4.2, "completion_rate": 85, "execution_frequency": "每天执行", "action_plan": "制定每日学习计划，提升专业技能", "reason": "通过持续学习来提高工作竞争力" } }
📥 返回示例
{ "success": true, "message": "进展反馈生成成功", "data": { "status_feedback": [ { "emoji": "📈", "title": "执行频率表现", "content": "根据当前数据，你的执行频率处于良好水平..." } ], "improvement_suggestions": [ { "emoji": "🎯", "title": "提升执行频率", "content": "建议制定更规律的执行计划..." } ] } }
🧪 在线测试
POST /api/create-feishu-record
创建飞书表格记录，将文本内容保存到指定的行ID中
📤 请求示例
{ "text": "这是要保存的文本内容", "line_id": "行标识符" }
📥 返回示例
{ "success": true, "message": "飞书记录创建成功", "data": { "text": "这是要保存的文本内容", "line_id": "行标识符", "record_id": "rec123456" } }
🧪 在线测试
POST /api/get-table-column-stats
统计飞书表格中指定列的非空数据数量，获取列数据详情和统计信息
📤 请求示例
{ "column_id": "fldXXXXXXXXXXXXXX" }
📥 返回示例
{ "success": true, "message": "统计完成", "data": { "column_id": "fldXXXXXXXXXXXXXX", "count": 15, "data": [ { "record_id": "recXXXXXXXXXXXXXX", "value": "数据内容1" } ], "total_records": 20 } }
🧪 在线测试
GET /api/get-all-table-data
获取飞书表格的完整数据，包含所有记录和字段信息
📤 请求示例
GET /api/get-all-table-data 无需请求体
📥 返回示例
{ "success": true, "message": "获取成功", "data": { "records": [ { "record_id": "recXXXXXXXXXXXXXX", "fields": { "字段1": "值1", "字段2": "值2" } } ], "total_records": 50 } }
